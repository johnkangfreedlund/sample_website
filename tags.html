Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1‐h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div division indicate division in text
header header describe text as header
ol ordered list populate numbered list
ul unordered list populate bulleted list

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make emoboldened text <strong>technical sophistication</strong> technical sophistication
a anchor link to another location via text <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code display stylized inline elements <code>code<code> code
span span indicate span in text <span>Melville's novel</span> Melville's novel
li list Create appropriate list item <ol><li>one</li><li>two</li></ol>
  1. one
  2. two

Documents Tags

Tag Name Purpose
html HTML indicate HyperText Markup Language file
head header indicate title and other key details
body body establish body of document
title title Supplies title for tab and other, non-visual uses
meta meta indicate important page information, such as character set

Further Tags

Tag Name Purpose
br line break Inserts a single line break
div division Defines a division or a section in an HTML document
span span Used to group inline-elements in a document
blockquote blockquote Specifies a section that is quoted from another source
header header Represents a container for introductory content or a set of navigational links
